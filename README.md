# 英雄联盟英雄介绍站

一套完整的英雄联盟(Legend of Legends)游戏介绍网站，突出每个英雄的详细玩法攻略。

## 🎮 网站功能

### 1. 首页
- 游戏特色介绍
- 五大位置角色说明（上单、打野、中单、射手、辅助）
- 热门英雄展示
- 游戏统计信息

### 2. 英雄列表页
- 完整的英雄数据库（包含16位热门英雄）
- 多维度筛选功能：
  - 按角色类型筛选（战士、刺客、法师、射手、辅助、坦克）
  - 按难度筛选（简单、中等、困难）
  - 搜索功能
- 英雄卡片展示基础信息

### 3. 英雄详情页
每位英雄包含完整信息：
- **基础信息**：名称、称号、角色定位、背景故事 + Emoji头像
- **技能介绍**：被动 + Q/W/E/R四个主动技能，包含冷却时间和消耗
- **玩法攻略**：
  - 前期对线策略
  - 中期团战思路
  - 后期团战定位
- **出装推荐**：核心装备 + 可选装备
- **符文推荐**：主系 + 副系符文搭配
- **皮肤系统**：每位英雄5个皮肤，支持点击查看详情
- **相关英雄推荐**：同类型的其他英雄

### 4. 游戏模式页
- 召唤师峡谷（经典5v5）
- 极地大乱斗（ARAM）
- 无限火力（URF）
- 云顶之弈（自走棋）
- 训练模式等

### 5. 新手指南页
- 游戏基础介绍
- 游戏流程详解（前期/中期/后期）
- 重要资源说明（元素龙、峡谷先锋、大龙等）
- 新手建议（6条核心提示）
- 新手推荐英雄

## 🛠 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 现代样式设计，包含：
  - CSS Variables（主题色彩管理）
  - Flexbox & Grid 布局
  - 响应式设计（移动端适配）
  - 动画效果
- **JavaScript (ES6+)** - 交互逻辑：
  - 数据动态加载
  - 筛选和搜索功能
  - 页面路由处理
  - 滚动动画

## 📁 项目结构


## 🚀 如何运行

### 方法1：直接打开
1. 进入项目目录
2. 直接用浏览器打开 `index.html` 文件

### 方法2：本地服务器（推荐）
使用Python启动本地服务器：

```bash
# Python 3
cd league-of-legends-website
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后访问：`http://localhost:8000`

### 方法3：VS Code Live Server
1. 在VS Code中安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🔧 自定义扩展

### 添加新英雄
编辑 `data/champions.json` 文件，按照现有格式添加新英雄数据：

```json
{
  "id": "hero-id",
  "name": "英雄名称",
  "title": "称号",
  "roles": ["角色1", "角色2"],
  "difficulty": "简单/中等/困难",
  "lore": "背景故事...",
  "abilities": {
    "passive": {...},
    "q": {...},
    "w": {...},
    "e": {...},
    "r": {...}
  },
  "playstyle": {
    "early": "前期策略...",
    "mid": "中期策略...",
    "late": "后期策略..."
  },
  "builds": {
    "core": ["装备1", "装备2", "装备3"],
    "situational": ["可选装备1", "可选装备2"]
  },
  "runes": {
    "primary": "主系名称",
    "keystone": "基石符文",
    "secondary": "副系名称"
  }
}
```

### 添加英雄图片
1. 将英雄头像图片放入 `images/heroes/` 目录
2. 在 `createChampionCard` 函数中修改图片路径
3. 建议图片尺寸：300x300像素，PNG或JPG格式

### 皮肤系统
每个英雄都配备了完整的皮肤数据，包含：
- **5个皮肤**：经典 + 4个付费皮肤
- **皮肤等级**：经典、普通、史诗、传说、限定
- **价格体系**：免费（经典）、750 RP（普通）、975 RP（史诗）、1350/1820 RP（传说）
- **交互功能**：点击皮肤缩略图可查看详情面板

皮肤数据结构示例：
```json
"skins": [
  {
    "id": "aatrox-classic",
    "name": "暗裔剑魔",
    "price": "免费",
    "tier": "经典",
    "description": "亚托克斯的原始形态"
  },
  {
    "id": "aatrox-odyssey",
    "name": "奥德赛",
    "price": "1820 RP",
    "tier": "传说",
    "description": "来自奥德赛宇宙的星际战士"
  }
]
```
